5、启动、关闭

6、redis集群redis-cluster

最好至少有六个redis才能搭建集群，让其两两一对形成主备关系。因为集群各个节点的数据不一样，所以需要主备才可以实现，高可用。否则一个节点挂了，数据就不完整了。

集群一共有16384个槽，每个节点指定一定范围的槽，redis会计算数据具体存在哪个节点中。
